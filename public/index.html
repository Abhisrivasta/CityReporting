About This Project

The Smart City API allows citizens to report city issues like potholes, garbage, water leaks, and more. Admins can track and resolve these reports efficiently.

Key Features

  • User authentication & authorization
  • Report issues with images & location
  • Cloudinary image storage
  • Role-based access for admins & users
  • Real-time updates for report status
